# MmmDining
### University of Michigan Dining on your (WearOS) wrist!

Check dining menus at lightning speed from your wrist and impress all your friends!

I put this together in an afternoon after getting tired of fishing out my phone to figure out which Michigan dining hall has the best food. Thus, MmmDining was born!

The app is super simple, and fetches real-time menus from the Michigan Dining API (the same API used by the official Michigan app). Figured I'd put it on GitHub in case
one of you happens to have a Wear OS watch :)

APKs are in the releases section if you'd like to give the app a whirl yourself.
